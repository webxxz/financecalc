Calculation Method

This calculator uses an iterative method (Newton-Raphson approximation) to find the interest rate that satisfies the EMI formula.

EMI = [P x R x (1+R)^N] / [(1+R)^N-1]

Where:

  • P = Principal Loan Amount
  • R = Monthly Interest Rate
  • N = Number of Monthly Installments
