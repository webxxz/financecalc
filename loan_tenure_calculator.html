Calculation Method

This calculator estimates the loan tenure using the EMI formula:

EMI = [P x R x (1+R)^N] / [(1+R)^N-1]

Where:

  • P = Principal Loan Amount
  • R = Monthly Interest Rate
  • N = Number of Monthly Installments (Tenure)
